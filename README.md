# 🇮🇳 India Monitor

A real-time India dashboard tracking NIFTY, SENSEX, top stocks, gold, crude oil, forex rates, crypto in ₹, weather for 4 major cities, and news from NDTV, ET & The Hindu. No backend, no API keys — runs in the browser and auto-refreshes every 60 seconds.

**Live site:** https://amanpal-tech.github.io/india-monitor/

## What it tracks

| Section | Data |
|---|---|
| 📈 Markets | NIFTY 50, SENSEX |
| 🏢 Stocks | Reliance, TCS, HDFC Bank, Infosys, Wipro |
| 💱 Forex | USD, EUR, GBP, JPY, AED vs ₹ |
| ₿ Crypto | Bitcoin, Ethereum, BNB, XRP in ₹ |
| 🪙 Commodities | Gold, Silver, Crude Oil (converted to ₹) |
| 🌤 Weather | Delhi, Mumbai, Bengaluru, Chennai |
| 📰 News | NDTV India · Economic Times · The Hindu |

## Tech

Plain HTML/CSS/JS — single file, no framework, no build step. Data from Yahoo Finance, CoinGecko, Frankfurter, Open-Meteo, and RSS feeds.
